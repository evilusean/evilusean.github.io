How to Use

  1. Upload Images: Click the upload area or drag and drop one or multiple images
  2. View Metadata: The app will automatically extract and display metadata from each image including:
    • File information (name, size, type)
    • Image dimensions and technical details
    • EXIF data (camera, GPS, date taken, etc.)
    • IPTC/XMP data (keywords, copyright, creator info)
  3. Download: Click "Download Metadata as Text" to save all metadata to a .txt file

Privacy Note: All processing happens in your browser. No data is uploaded to any server.

Need More Metadata? This browser-based tool extracts most common metadata fields. For comprehensive metadata extraction including all proprietary maker notes and advanced fields, use exiftool on Linux/Mac:
exiftool your-image.jpg
Install with: sudo apt install libimage-exiftool-perl (Debian/Ubuntu) or brew install exiftool (Mac)
