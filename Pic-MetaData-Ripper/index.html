How to Use

  1. Upload Images: Click the upload area or drag and drop one or multiple images
  2. View Metadata: The app will automatically extract and display metadata from each image including:
    • File information (name, size, type)
    • Image dimensions
    • EXIF data (camera, GPS, date taken, etc.)
  3. Download: Click "Download Metadata as Text" to save all metadata to a .txt file

Privacy Note: All processing happens in your browser. No data is uploaded to any server.
