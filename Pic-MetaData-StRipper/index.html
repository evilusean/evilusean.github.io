Supported Formats & Usage

How to Use

  1. Drag & drop images or click the drop zone to select files
  2. Wait for processing (metadata is automatically stripped)
  3. Optionally rename files and change output format
  4. Download individually or all at once as a ZIP file

Input Formats

  • JPEG/JPG - Standard photo format
  • PNG - Graphics with transparency
  • WebP - Modern web format
  • BMP - Bitmap images
  • GIF - Animated/static images (converts to PNG)
  • AVIF - Next-gen format (browser support varies)
  • TIFF/TIF - High-quality images (converts to PNG)

Output Formats

  • JPEG - Best for photos, smaller file size
  • PNG - Best for graphics with transparency
  • WebP - Modern format with great compression
  • BMP - Uncompressed bitmap
  • AVIF - Next-gen format (Chrome/Edge only)

What Gets Removed

  • EXIF data (camera settings, timestamps)
  • GPS coordinates (location data)
  • Copyright information
  • Camera make and model
  • Software/editing history
  • Embedded thumbnails
